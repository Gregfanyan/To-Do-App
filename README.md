## General info

A simple todo application bootstrapped by webpack that features CRUD functions


## Technologies

-   React.js
-   TypeScript
-   Classnames
-   Local Storage
-   SCSS
-   Webpack
-   Babel
-   UUID

## Setup

Feel free to clone the app, install with the command `npm install` or `yarn install ` and run `npm start` or `yarn run`
